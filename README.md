# Log Analyzer

A Python CLI tool to analyze log files for suspicious activity. Designed for cybersecurity professionals and system administrators.

## Features
- Parse and analyze log files (Apache, Nginx, syslog, etc.)
- Detect suspicious patterns (failed logins, brute force, etc.)
- Generate summary reports
- Easy to extend with custom rules

## Requirements
- Python 3.8+
- No external dependencies (uses Python standard library)

## Usage
1. Clone the repository or download the source code.
2. Run the analyzer on a sample log file:
   ```sh
   python log_analyzer.py sample.log
   ```
3. View the generated report in the terminal.
